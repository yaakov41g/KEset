How to use :

Choose languages from the list. / at least two languages /

To correct simple letters press 'ctrl' + the key under blue arrow (mostly '<' key).

To correct capslock press 'ctrl' + the key under green arrow (mostly '>' key).

To enforce uniform conversion press 'ctrl' + '|' keys. / see description below /

Instructions

* With this application you can correct a text that was typed in a wrong language (gibberish).
* Select 2 languages at least. Those whom you use.
* In order to correct simple letters (that is , not BIG letters) , use 'Ctrl' + '<'(Less-Than sign). Make sure the text element is focused.
* If you typed wrongly when Capslock key is on, (BIG LETTERS) , correct this by pressing 'Ctrl'(control key) + '>'(Greater-Than sign).
* If you have a text with a mix of languages to convert , you can enforce one-language conversion by 'Ctrl' + '|'(vertical bar).
* If after conversion for some reason you need back the original wrong text , but it was erased , you can get the original text of the last 3 conversions , in the little rectangle(text area) above. I recommend , that if the content you typed is very important for you , copy it before converting. If you converted a secret code like password , you should convert a new text (or empty) 3 more times untill it disappears.
* On Facebook/Twitter you can get the converted text by clicking the plugin icon, in the opened pop up window.

* This is a first version of the application.
