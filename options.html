Description for the application

* By this application you can correct a text that was typed in wrong language (gibberish).
* You should select the languages that you use of , those you may make typing mistake with them.
* Select 2 languages at least.
* In order to correct simple letters (that is , not big letters) , use 'Ctrl' + '<'(left square bracket key). Make sure the text element is focused. * If you typed wrongly when Capslock key is on, (BIG LETTERS) , correct this by pressing 'Ctrl'(control key) + '>'(right square bracket key).
* If you need to convert a line with a mix of languages , you can enforce one-language convertion by 'Ctrl' + '|'(vertical bar).
* If for some reason you need back the original wrong text , but it was erased , you can get the original text of the last convertion , in the little rectangle(text area) above. I recommend , that if the content you typed is very important for you , copy it before converting.
* In this text area you can also make converting in case it was not possible on the page.
* This is a beta version of the application.
* I wish you happy and correct typing.
